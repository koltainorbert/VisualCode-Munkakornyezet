$temaMappa = (Get-Location).Path

$utolso = ""
while ($true) {
    $most  = Get-Date
    $datum = $most.ToString("yyyy. MMMM dd.")
    $ido   = $most.ToString("HH:mm")
    $fajlok = @(git diff --name-only HEAD 2>$null) + @(git ls-files --others --exclude-standard 2>$null) | Where-Object { $_ -ne "" }
    if ($fajlok.Count -gt 0) {

        if ($sor -ne $utolso) {
            $utolso = $sor
            $tartalom = Get-Content $naplo -Raw -Encoding UTF8
            if ($tartalom -match [regex]::Escape("## $datum")) {
                $tartalom = $tartalom -replace "(?m)(## $([regex]::Escape($datum))[\s\S]*?)(---)", "`$1- $sor`n`$2"

                $tartalom += "`n## $datum`n`n### Valtozasok:`n- $sor`n`n---`n"
            }
            Set-Content $naplo $tartalom -Encoding UTF8
            git add .
            git diff --cached --quiet
            if ($LASTEXITCODE -ne 0) {
                git commit -m "Auto $ido — $($fajlok -join ', ')"
                git push
                Write-Host "[$ido] Pushed: $($fajlok -join ', ')" -ForegroundColor Green
            }
        }

        Write-Host "[$ido] Nincs valtozas." -ForegroundColor DarkGray

`$naplo = "$projektMappa\NAPLO.md"
`$ido = Get-Date -Format 'HH:mm:ss'
`$fajlok = @(git diff --name-only HEAD 2>`$null) + @(git ls-files --others --exclude-standard 2>`$null) | Where-Object { `$_ -ne "" }
if (`$fajlok.Count -gt 0) {
    git add .
    git diff --cached --quiet
    if (`$LASTEXITCODE -ne 0) {
        git commit -m "AutoSave `$ido — `$(`$fajlok -join ', ')"
        git push 2>`$null
    }
}
"@ | Set-Content $scriptPath -Encoding UTF8

$taskName = "SDH-AutoSave-$(Split-Path $projektMappa -Leaf)"
Start-Process powershell -Verb RunAs -ArgumentList @"
-ExecutionPolicy Bypass -Command "
Unregister-ScheduledTask -TaskName '$taskName' -Confirm:`$false -ErrorAction SilentlyContinue;
`$a = New-ScheduledTaskAction -Execute 'powershell.exe' -Argument '-ExecutionPolicy Bypass -WindowStyle Hidden -File ""$scriptPath""';
`$t1 = New-ScheduledTaskTrigger -AtLogOn;
`$t2 = New-ScheduledTaskTrigger -RepetitionInterval (New-TimeSpan -Minutes 5) -Once -At (Get-Date);
`$s = New-ScheduledTaskSettingsSet -StartWhenAvailable -ExecutionTimeLimit (New-TimeSpan -Minutes 2);
Register-ScheduledTask -TaskName '$taskName' -Action `$a -Trigger `$t1 -Settings `$s -RunLevel Highest -Force;
Write-Host 'Task beregisztralva!'; Start-Sleep 2"
"@ -Wait -WindowStyle Normal

on:
  push:
    branches: [ main ]

  deploy:

01 Rendszer célja és koncepciója

A VS Code Munkaközeg Wizard egy interaktív PowerShell-alapú rendszer, amelynek célja, hogy bármilyen típusú fejlesztőprojekthez egyetlen parancs futtatásával felállítson egy teljes, profi munkaközeg-infrastruktúrát. A fejlesztőnek nem kell manuálisan konfigurálni a VS Code taszkokat, a Git repót, az auto-save folyamatokat, a napló rendszert vagy a CI/CD pipeline-t.

A wizard egyszer kérdez, aztán mindent automatikusan készít el:

  • VS Code feladatok (tasks.json) — Push, Pull, Visszagörgetés, Napló, Browser Sync, Branch kezelés
  • Auto-mentés háttérfolyamat — Windows Task Schedulerben fut, 15 percenként automatikusan commitol
  • Napló watcher — FileSystemWatcher alapú, minden mentéskor frissíti a NAPLO.md-t
  • GitHub Copilot instrukciók — projektspecifikus AI utasítások (.github/copilot-instructions.md)
  • Deploy pipeline — opcionális GitHub Actions FTP-deploy workflow

A rendszer mottója: "Egy parancs — minden kész." Új gépen, új projektnél: beilleszted a varázs-parancsot a PowerShell-be, válaszolsz a kérdésekre, és teljes értékű munkaközeg áll rendelkezésre.

05 VS Code Taskok részletesen

Task neveMit csinál pontosanMikor használd
🔴 Git Push Bekéri a commit üzenetet, majd: git add . → git commit -m "..." → git push origin [branch] Manuális mentésnél — napi munka végén, fontos mérföldköveknél, release előtt
⬇ Git Pull git pull origin [branch] — letölti a remote repository legújabb változásait Más gépen vagy kollégával végzett munkánál, munka elején szinkronizáláshoz
↩ Visszagörgetés git reset --soft HEAD~1 — az utolsó commitot visszavonja, a fájlok változatlanok maradnak, csak az index törlődik Ha rossz commit üzenetet adtál, vagy az utolsó commitot módosítani szeretnéd
📝 Napló Watcher Elindítja a scripts/naplo-watcher.ps1-t egy új terminálban — FileSystemWatcher figyeli az összes fájlváltozást Munka elején indítsd el — terminálablakot nyit, folyamatosan fut a háttérben
🌐 Browser Sync npx browser-sync start --server --files "**/*" — helyi fejlesztői szerver + automatikus böngésző-reload fájlváltozásnál HTML/CSS/JS fejlesztésnél — böngészőben élőben látod a változásokat mentés után
🌿 Új Branch Bekéri a branch nevét, majd: git checkout -b [nev] → git push -u origin [nev] Új funkció fejlesztésekor — ne main-en dolgozz közvetlen, használj feature branch-et

5

Másold be és futtasd a PowerShell terminálba (nem Win+R, nem CMD!):

⚠ VS Code terminálban irm | iex NEM működik — az interaktív kérdések nem jelennek meg! Helyette:

irm "https://raw.githubusercontent.com/koltainorbert/VisualCode-Munkakornyezet/main/setup-projekt.ps1" -OutFile "$env:TEMP\setup-projekt.ps1" ; powershell -ExecutionPolicy Bypass -File "$env:TEMP\setup-projekt.ps1"

Win+R → standalone PowerShell ablakban az irm ... | iex működik.
