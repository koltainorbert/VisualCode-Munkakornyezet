>> Projekt neve

Projekt neve

>> Mappa

Pl: C:\projektek\mywebshop – ha nem létezik, létrehozza. Default: aktuális mappa.

>> GitHub URL

GitHub repo URL

>> Tipus [2]

Projekt típusa

[1] WordPress  [2] HTML/CSS/JS  [3] React/Vue  [4] Node.js  [5] Laravel  [6] Egyéb
Browser Sync beállítása típus szerint változik (proxy vs. server mód).

>> Proxy URL

Browser Sync proxy (WordPress/Laravel esetén)

WordPress: myproject.local – HTML esetén automatikusan --server mód, nem kell megadni.

>> Port [3000]

Browser Sync port

Alapértelmezett: 3000. Ha más portot használ valamid, írj mást.

DEFAULT: 3000

deploy.yml template

FTP auto-deploy

A wizard generálja ezt – a 4 GitHub Secret beírása után automatikus az éles deploy.

.github/workflows/deploy.yml
name: Deploy - Eles szerver

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: FTP Deploy
        uses: SamKirkland/FTP-Deploy-Action@v4.3.4
        with:
          server: ${{ secrets.FTP_SERVER }}
          username: ${{ secrets.FTP_USERNAME }}
          password: ${{ secrets.FTP_PASSWORD }}
          server-dir: ${{ secrets.FTP_SERVER_DIR }}
          exclude: |
            **/.git*
            **/.git*/**
            **/node_modules/**
            **/.vscode/**
