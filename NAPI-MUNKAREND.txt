===========================================
  NAPI FEJLESZTÉSI MUNKAREND
===========================================

REGGEL (projekt megnyitáskor):
  1. VS Code megnyitás → project mappa
  2. Napló Watcher automatikusan elindul (Allow ha kér)
  3. Ha kell: Ctrl+Shift+P → Run Task → ⬇ Pull

KÖZBEN (fejlesztés):
  - Browser Sync: localhost:3000 → azonnali frissülés
  - Ctrl+Shift+B → push ha manuálisan akarod
  - Auto: minden 1 percben (watcher) + 5 percben (Task Scheduler)

ESTE (befejezéskor):
  - Ctrl+Shift+B → végső push
  - NAPLO.md automatikusan frissült

HA ELROMLOTT VALAMI:
  Ctrl+Shift+P → Run Task → ↩ Visszagörgetés
  → Listázza az utolsó 20 mentési pontot
  → Visszaállítja a kiválasztott verzióra

HASZNOS PARANCSOK:
  git log --oneline -10    # utolsó 10 commit
  git status               # mi változott
  git diff                 # pontosan mi változott
